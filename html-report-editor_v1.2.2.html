About this editor
Visual Report Editor

Open an AI-generated HTML report and edit it like a page — no code required. Changes are written straight into the report's own HTML/CSS and saved back to the file.

What you can do
  • Style mode — click any element to select it, then change colors, fonts, size, spacing, width, borders, shadows and more.
  • Edit text — double-click any text and type to rewrite it.
  • Move & reorder — nudge a box, move it up/down, or drag it among its neighbors.
  • Insert & theme — add ready-made blocks, apply KF brand themes, and recolor to the palette.
  • Save — writes back to the original file (Edge/Chrome) or downloads an edited copy.
Keyboard shortcuts

Ctrl+Z undo · Ctrl+Y redo · Ctrl+S save · arrow keys nudge (Shift = 10px) · Delete removes the selected element.

Good to know
  • It may not work with every report — unusual structures or scripts can limit what's editable.
  • Not every element can be selected or edited individually.
  • Big layout or position changes can affect how the report responds on other screen sizes — prefer small adjustments, and check the result before sharing.
  • Use Revisions to snapshot a good state before larger changes, and Save a copy… rather than overwriting an original you might still need.
Privacy
Runs entirely in your browser. Reports are never uploaded — nothing leaves your machine, and no internet connection is used.
