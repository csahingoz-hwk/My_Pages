How this works

A Power Automate flow (part of your KornFerry M365 license) runs every morning, reads your unread emails, classifies them, and saves a JSON report to your OneDrive. You open this page, load that file, and see your inbox classified at a glance — no Azure registration, no IT approval needed.

✅ Power Automate is included in every Microsoft 365 Business / Enterprise license. You can access it at make.powerautomate.com with your KornFerry account.

Step-by-step Power Automate setup

  1. Go to make.powerautomate.com and sign in with your KornFerry account.
  2. Click Create → Scheduled cloud flow. Set it to run daily at 8:00 AM (or whenever you start work).
  3. Add the first action: search for Office 365 Outlook – Get emails (V3).
    Configure it:
    • Folder: Inbox
    • Fetch Only Unread Messages: Yes
    • Include Attachments: No
    • Top: 50
    • Received Time: set Filter Query to receivedDateTime ge '@{startOfDay(utcNow())}'
  4. Add action: Data Operation – Select. Map the following fields from the email output:
    "subject": item()?['subject']
    "from": item()?['from']
    "receivedTime": item()?['receivedDateTime']
    "bodyPreview": item()?['bodyPreview']
    "importance": item()?['importance']
    "hasAttachments": item()?['hasAttachments']
  5. Add action: Data Operation – Compose. Set Inputs to:
    outputs('Select')
  6. Add action: OneDrive for Business – Create file.
    • Folder Path: /Email Reports (create this folder in your OneDrive)
    • File Name: email-report-@{formatDateTime(utcNow(),'yyyy-MM-dd')}.json
    • File Content: use the Outputs from the Compose step
  7. Click Save then Test → Manually to run it now. Open your OneDrive → Email Reports folder — you should see a .json file.
  8. Download that JSON file and load it in the Load Report tab above. Done!

Alternative: Run it instantly (no schedule needed)

Instead of a scheduled flow, create an Instant cloud flow with a manual trigger button. Then tap it from the Power Automate mobile app whenever you want a fresh report. The rest of the steps are identical.
